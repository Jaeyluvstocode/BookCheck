## Structure
/server - backend (Node/Express/MongoDB)
/client - frontend (React)

## Run locally
1. Start MongoDB locally or use a cloud MongoDB and set MONGO_URI in server/.env
2. Start backend:
   cd server
   npm install
   copy .env.example to .env and fill values
   npm start
3. Start frontend:
   cd client
   npm install
   npm start
